# Todo app

> Simple WebApp to manage daily tasks

## Built With

- React
- Typescript
- Tailwind
- Shadcn UI

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- node and npm installed

### Setup

- clone the repo.
- run npm i to install packages.

### Running

- npm run dev
